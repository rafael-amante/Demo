Introdução

JavaScript é uma linguagem de programação amplamente usada para desenvolver páginas de web interativas. Criada em 1995, é uma linguagem de alto nível, dinâmica e baseada em eventos, permitindo manipular o conteúdo HTML e CSS de forma eficiente. Atualmente, é utilizada tanto no frontend (com frameworks como React e Vue.js) quanto no backend (com Node.js). A sua versatilidade e ampla adoção a tornam essencial para o desenvolvimento web moderno.
